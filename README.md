# CarSeller

## Описание на проекта

**CarSeller** е уеб приложение за покупко-продажба на автомобили.

## Технологии

Проектът е разработен, като се използва основно PHP за логиката на сървъра и CSS за стилизиране на потребителския интерфейс.

* **Backend:**
    * PHP
    * База данни: MySQL 
* **Frontend:**
    * HTML5
    * CSS3
    * JavaScript (малко)
